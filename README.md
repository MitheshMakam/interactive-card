# 🍰 Interactive Recipe Card with Animations

A responsive and interactive **Recipe Card Web Application** built using **HTML, CSS, and JavaScript**.  
This project demonstrates modern front-end development concepts including layout design, CSS animations, responsiveness, and DOM-based interactivity.

---

## 📌 Project Objective

To design an interactive recipe card where users can:

- View a complete list of ingredients
- Follow step-by-step cooking instructions
- Check preparation and cooking time
- Experience smooth UI animations
- Interact dynamically using JavaScript

This project focuses on combining **UI design, responsiveness, animations, and basic scripting** into a single functional web application.

---

## 🚀 Key Features

- ✔️ Clean and modern recipe card UI
- ✔️ Structured ingredients section
- ✔️ Step-by-step cooking guide
- ✔️ Preparation & cooking time display
- ✔️ Smooth CSS animations & transitions
- ✔️ Fully responsive design (mobile + desktop)
- ✔️ Basic JavaScript interactivity

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & Animations  
- **JavaScript (ES6)** – Interactivity
